# PriceFit

> [🇬🇧 English](README.md) | 🇷🇺 Русский

Веб-приложение на Django для формирования рекомендаций по целевым группам на основе сопоставления ценовых диапазонов. Пользователи создают запросы с указанием диапазона цен; система вычисляет коэффициент корреляции Жаккара для каждой целевой группы и автоматически ранжирует результаты.

## Возможности

- **Аккаунты пользователей** — регистрация, вход и выход через встроенную аутентификацию Django
- **Запросы** — создание, просмотр, редактирование и удаление запросов с ценовым диапазоном
- **Целевые группы** — каталог групп с бюджетными диапазонами (управляется staff-пользователями)
- **Автоматический анализ** — одним нажатием вычисляет коэффициент Жаккара для каждой группы
- **Панель администратора** — полный доступ через Django admin
- **Адаптивный интерфейс** — Bootstrap 5 с визуализацией коэффициента в виде прогресс-бара

## Установка

```bash
# Клонировать репозиторий
git clone <repo-url>
cd Программа

# Создать и активировать виртуальное окружение
python -m venv .venv
source .venv/bin/activate      # Linux / macOS
.venv\Scripts\activate         # Windows

# Установить зависимости
pip install -r requirements.txt

# Применить миграции
python manage.py migrate

# Создать суперпользователя (необходим для управления целевыми группами)
python manage.py createsuperuser

# Запустить сервер разработки
python manage.py runserver
```

Приложение будет доступно по адресу `http://127.0.0.1:8000/`.

## Использование

1. Зарегистрируйте аккаунт или войдите в систему.
2. Перейдите в **Целевые группы** — staff-пользователь (или вы сами через панель администратора) добавляет группы с бюджетными диапазонами.
3. Создайте запрос: **Мои запросы → Новый запрос**, укажите диапазон цен.
4. Откройте запрос и нажмите **Запустить анализ** — система вычислит коэффициент (0–1) для каждой группы. Чем выше значение, тем лучше совпадение диапазонов.

### Интерпретация коэффициента

| Диапазон | Значение |
|---|---|
| 0.7 – 1.0 | Высокое соответствие |
| 0.3 – 0.7 | Частичное соответствие |
| 0.0 – 0.3 | Слабое или нулевое соответствие |

## Переменные окружения

Для продакшн-развёртывания задайте следующие переменные окружения вместо значений по умолчанию:

| Переменная | Описание | По умолчанию |
|---|---|---|
| `SECRET_KEY` | Секретный ключ Django | небезопасный ключ разработки |
| `DEBUG` | Режим отладки (`True`/`False`) | `True` |
| `ALLOWED_HOSTS` | Список допустимых хостов через запятую | _(пусто)_ |

## Маршруты URL

| Метод | URL | Описание |
|---|---|---|
| GET | `/` | Главная / дашборд |
| GET/POST | `/accounts/register/` | Регистрация |
| GET/POST | `/accounts/login/` | Вход |
| POST | `/accounts/logout/` | Выход |
| GET | `/requests/` | Список запросов пользователя |
| GET/POST | `/requests/new/` | Создать новый запрос |
| GET | `/requests/<id>/` | Детали запроса с рекомендациями |
| GET/POST | `/requests/<id>/edit/` | Редактировать запрос |
| POST | `/requests/<id>/delete/` | Удалить запрос |
| POST | `/requests/<id>/analyze/` | Запустить анализ |
| GET | `/groups/` | Список целевых групп (публичный) |
| GET/POST | `/groups/new/` | Создать группу (только staff) |
| GET/POST | `/groups/<id>/edit/` | Редактировать группу (только staff) |
| POST | `/groups/<id>/delete/` | Удалить группу (только staff) |
| GET | `/admin/` | Панель администратора Django |
